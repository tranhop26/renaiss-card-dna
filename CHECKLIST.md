## 🚀 Deployment Quick Guide

### Backend (Railway):
```bash
1. Push to GitHub
2. Connect Railway to repo
3. Auto-deploy
4. Get URL: https://your-app.railway.app
```

### Frontend (Vercel):
```bash
1. Push to GitHub
2. Connect Vercel to repo
3. Set Root Directory: "frontend"
4. Add env: NEXT_PUBLIC_API_URL=[railway-url]
5. Deploy
6. Get URL: https://your-app.vercel.app
```
