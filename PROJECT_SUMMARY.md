## 🚀 Để chạy dự án

### Backend (Terminal 1):
```bash
cd backend
pip install -r requirements.txt --break-system-packages
python main.py
```

### Frontend (Terminal 2):
```bash
cd frontend
npm install
npm run dev
```

### Truy cập:
- Frontend: http://localhost:3000
- Backend API: http://localhost:8000
- API Docs: http://localhost:8000/docs

## 🎯 Các tính năng đã hoàn thành

### ✅ Backend API (FastAPI)
1. **GET /api/cards** - Lấy danh sách tất cả cards
2. **GET /api/card/{id}/dna** - Phân tích DNA của card
3. **GET /api/collector/{wallet}/profile** - Profile collector
4. **GET /api/match/{wallet}/recommendations** - Gợi ý cards
5. **GET /** - Health check

### ✅ AI Engine
- Visual DNA Analysis (color, style, complexity)
- Behavioral DNA Analysis (trading patterns, collector types)
- Market DNA Analysis (volatility, momentum, synergy)
- Matching Algorithm (weighted scoring)
- Collector Profiling (deterministic generation)

### ✅ Frontend Pages
1. **Homepage** - Hero section với browser mockup
2. **Card Analyzer** - Analyze bất kỳ card nào
